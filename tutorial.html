Alleropedia

Comprehensive database for Allergens

This website and the database is developed considering the need of a common metadatabase that consolidates data from some existing databases on allergens with some additional information from different reputed journals and databases added manually and data processed computationally.
Additional features and functionalities have been added to make it easy to read, access, analyse and further use the data.

This section has been added to provide the users a complete knowledge for understandig the database and the applications features.

TUTORIAL

1. Information sources:

This database takes its major data from 4 databases:
  1. Allergen Nomenclature Database
  2. Allergen Online Database
  3. Comprehensive Protein Allergen Resource (COMPARE) Database
  4. Allergome Database

You can find information about them here
The EPPO Global Database , Invasive Species Compendium (CABI - Centre for Agriculture and Bioscience International) and NCBI resources have also been used for contributing the data to this database.

2. Columns in the database:

The present database stores its data in 18 different columns.
  • Binomial name of species
  • Common Name of allergen source
  • Biochemical name of allergenic proteins.
  • Authority
  • Habitat of allergen source
  • Organism Type of allergen source.
  • Molecular weight of allergenic proteins.
  • Description about allergenic proteins.
  • Different sources of allergen.
  • IUIS name of allergenic proteins.
  • Accession number of allergenic protein.
  • Length of allergenic protein.
  • Link to website.
  • Route of allergen exposure
  • Group (includes genus of source of allergen and allergenic protein)
  • Allergencity of allergenic proteins.
  • GeneID for organism.

You can get information about each of these columns here

3. Features offerred:

We have the following basic features to extract usable information from the database:
  1. Filter text for different fields

    This features allows users to display only limited data entries according to their selection of filters for different columns. Pattern matching instead of exact full text match is used, thus increasing the database usability.



    2. To filter text for different fileds,
    1. Enter the text in the different input boxes for the fileds you want.
    2. You can add single or multiple filters.
    3. Click on 'Search' Button.
    4. The data table gets modified according to the selected text filters.
    5. To reset, click on the 'Reset' button.



  2. Filter fields according to starting characters

    This feature allows users to selct a field and a letter from the drop down menus to display only those data rows which have the particular field starting with the specific letter only.



    3. To filter fields according to starting charaters,
    1. Select the column from the drop down menu
    2. Select the letter from the adjacent drop down menu.
    3. Click on 'Choose options' button.
    4. The data table gets modified with the chosen filed entries starting with the chosen letter



  3. Download the database.

    This feature allows users to download the complete database to access it later from their own systems or use it for their own research, analysis or any other purpose. The database gets downloaded in the form of an excel file.



    4. To download the database, click on 'Download' button
